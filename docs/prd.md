# Web Camara - Product Requirements Document (PRD)

## 1. 产品概述 (Product Overview)
**Web Camara** 是一款基于 Web 的相机应用，旨在通过模拟真实相机的操作体验，帮助摄影初学者理解和掌握摄影基础知识。核心功能包括模拟曝光三要素（ISO、光圈、快门速度）、焦段切换以及实时调色。

## 2. 目标用户 (Target Audience)
- 摄影初学者
- 对相机参数感兴趣但尚未购买相机的用户
- 想要练习手动挡拍摄的手机摄影用户

## 3. 核心功能 (Core Features)

### 3.1 取景器与实时预览 (Viewfinder & Live Preview)
- **实时画面**: 调用设备摄像头显示实时画面。
- **辅助线**: 提供九宫格、黄金螺旋等构图辅助线。
- **直方图**: 实时显示亮度直方图，辅助判断曝光。

### 3.2 曝光模拟 (Exposure Simulation)
- **ISO (感光度)**:
    - 调节范围: 100 - 6400
    - 效果: 增加 ISO 提高画面亮度，同时**必须模拟高 ISO 下的噪点 (Grain) 效果**。
- **Aperture (光圈)**:
    - 调节范围: f/1.4 - f/16
    - 效果: 调节画面亮度（模拟进光量）。*进阶: 结合深度估算模拟景深虚化 (可选)*。
- **Shutter Speed (快门速度)**:
    - 调节范围: 1/8000s - 1s
    - 效果: 调节画面亮度。**必须模拟慢速快门下的低帧率或拖影感**。
- **曝光补偿 (EV)**: 自动计算当前模拟参数下的 EV 值，提示过曝或欠曝。

### 3.3 焦段模拟 (Focal Length Simulation)
通过裁剪 (Digital Crop) 模拟不同镜头的视角：
- **24mm**: 广角 (默认/无裁剪)
- **35mm**: 人文视角
- **50mm**: 标准视角
- **85mm**: 人像特写

### 3.4 色彩与风格 (Color & Style)
- **胶片模拟**: 提供几种经典的胶片预设。
- **基础参数**: 对比度、饱和度、色温调节。

### 3.5 教学模式 (Educational Mode)
- **被动式提示**: 用户调节 ISO/光圈/快门时，在控制区下方或附近显示简短的文字提示，解释该参数对画质、景深、运动捕捉的影响。不需要专门的教程模式。

## 4. 用户界面设计 (UI/UX Design)
- **设计语言**: **现代极简 (Modern Minimalist)**，参考 **Zed Editor** 的设计风格。
    - **主题**: 支持 **Light/Dark Mode**，默认跟随系统设置。
        - **Dark**: Zinc/Slate 深色系，专业沉浸。
        - **Light**: 清晰明亮，高对比度。
    - **字体**: 等宽字体 (Monospace)。
    - **元素**: 线条分隔，直角设计，无多余装饰。
    - **配色**: 高对比度，专业感。
- **响应式布局**:
    - **移动端 (Mobile)**: 类似 iOS 系统相机。主界面为全屏或大比例取景框，参数调节和快门按钮位于取景框下方。
    - **桌面端 (Desktop)**:
        - **中心**: 模拟手机比例的取景框。
        - **右侧**: 独立的参数调整面板 (ISO, 光圈, 快门, 焦段等)。

## 5. 技术栈 (Tech Stack)
- **前端框架**: SvelteKit
- **样式**: TailwindCSS (配合自定义配置实现 Zed 风格)
- **图像处理**: HTML5 Canvas / CSS Filters / WebGL (用于噪点和帧率模拟)
- **摄像头调用**: `navigator.mediaDevices.getUserMedia`

## 6. 迭代计划 (Roadmap)
- **Phase 1 (MVP)**: 项目搭建，基础取景，Zed 风格 UI 框架，曝光三要素模拟 (含噪点/低帧率)。
- **Phase 2**: 焦段裁剪，直方图，更多胶片滤镜。
- **Phase 3**: 拍照保存，画廊查看。
